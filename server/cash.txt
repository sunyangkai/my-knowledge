
造假动机
    保证上市或再融资
        上市必须有三年盈利，一旦成功上市就能募集巨额资金
    市值管理，避免退市
        上市公司扣非净利润为负且营收低于1亿元，将被ST，连续两年则终止上市。 管理层可以连续两年大亏第三年赚钱来钻这个漏洞。
    获取银行信用、达到贷款标准
        粉饰财务报表，故意虚增收入，虚高利润。
    完成对赌协议
        一家公司在收购另一家公司时，被收购方会做出业绩承诺。


造假思路：
    资产=负债+所有者权益，虚增了所有者权益，为了使报表平衡，要么在资产负债表中虚增资产，要么在资产负债表中虚减负债
    调高利润，就是费用资产化，或虚增收入；调低利润，就是资产费用化，或收入负债化

存货
    1.应该计提跌价而不计提。
        首先会计准则不能随便改。其次对比同行计提跌价的规则，如果宽松很多就有问题。
    2.虚构采购流出资金，再通过购买公司的产品增加营业收入。







利润创造的过程：
    营业利润 = 经营利润 + 【投资收益 + 公允价值变动收益 + 汇兑收益】 

    经营利润 = 营业总收入 - 营业总成本 + 【信用减值损失 + 资产减值损失 + 资产处置收益】

    营业总收入 = 营业收入 + 【利息收入 + 已赚保费 + 手续费及佣金收入】

    营业总成本 = 营业成本 + 【利息支出 + 手续费及佣金支出等】 + 【销售费用 + 管理费用 + 财务费用】
    // 注意，这里的利息支出已经相关的佣金是跟营业相关的利息收支，和财务费用里的利息收支不同，财务费用里的利息收支是和经营业务无关的。比如公司长期融资的债券支付的利息，公司多余现金用来购买债券产生的利息。

    营业成本 = 销售出去的存货价值


净利润：
    净利润 = 营业利润 + 【营业外收入 - 营业外支出】 - 所得税费用

    营业外收入：
        非流动资产处置利得、非货币性资产交换利得、出售无形资产收益、债务重组利得、企业合并损益、盘盈利得、债权人放弃的债权、政府补助、罚款收入、捐赠利得 等

    营业外支出：
         非流动资产处置损失、非货币性资产交换损失、债务重组损失、捐赠支出，行政罚款、固定资产报废、损毁、盘亏等
    所得税费用：


营业收入增长来源：
    1.行业增长
    2.市场份额扩大
    3.价格提升





净利润含金量  = 经营活动现金流金额 / 净利润。   净利润含金量大于1是好的， 小于1需要警惕
虚增利润不产生实际的现金流，付出的代价是多交税。上下游供应商也无法长期不收现金来供货。

毛利率：
   毛利率 = （营业收入 - 营业成本）/ 营业收入 
   毛利率衡量公司产品和服务端竞争力。高毛利代表竞争力强。低毛利的公司风险大，很容易被竞争对手取代。
   一般来说，毛利高于40%的企业比较有竞争优势。毛利在20% - 40%的企业存在竞争压力。毛利低于20%的，竞争压力大，风险高。建议高手投资
费用率：
    费用率 = （销售费用 + 管理费用 + 财务费用）/ 总营业收入  // 如果保守一些，财务费用为负数的时候就不加财务费用了
    费用过高和费用剧烈变化都是需要高度关注的。

    费用/毛利润。 这个比率 小于30%算优秀。 30% - 70%算一般。大于70%为恶劣

    销售费用过高，产品的吸引力和传播力差。销售费用过高的企业，在扩张过程中不仅需要扩大产品的生产能力，还需要配套新的团队，销售方案，资金。风险会逐渐扩大。
    管理费用增长速度应该小于等于营业收入增长幅度。增长需要匹配营业收入的增长幅度，不能剧烈波动。

竞争分析：
    1.供应商议价能力
        供应商的财务状况
             如果供应商处于财务困境中，他们将寻求任何机会来增加业务。因此，他们可能会在价格上做出让步，以维持与公司的合作关系
        供应商与公司的关系
            如果公司已经建立了长期、良好的合作关系，则供应商愿意在议价方面做出妥协，以保持良好的关系
        集中度
            供应商所在行业集中度越高，其议价能力越强。
            如果市场上存在多个供应商，他们之间的竞争将会降低每个供应商的议价能力。同样地，如果供应商独一无二或极少数，那么他们的议价能力就会增加
            行业集中度分析？
        供应商产品独特性：
            该物品是公司的核心零部件或关键原材料，那么供应商会利用自己的地位来谈判更有利的价格和条款
        替代性
            和行业集中度也有关。供应商所在行业行业分散，替代性越强。
        转换成本
            采购成本：与新供应商的谈判和合同签署成本。
            供应链重组成本：包括库存管理、供应链调整和仓储费用。
            质量控制和认证成本：确保新供应商符合质量标准和认证要求。
            培训和人员调整成本：为了适应新供应商的需求，可能需要培训员工或进行人员调整。
            技术和系统集成成本：如果新供应商使用不同的技术或系统，可能需要额外的投资。
            转换期间的停机时间成本：生产线停机或减产可能导致生产能力下降，损失产能。
            供应链中断风险：新供应商可能会引入风险，导致供应链中断，需要考虑这些风险的成本。
        资产负债表中垫付资金越多，供应商议价能力越强。
    2.客户议价能力
        竞争环境：
            市场上竞争对手的数量和性质会影响客户的议价能力。在竞争激烈的市场中，客户通常更容易谈判价格，因为他们有更多的选择。
        供应市场的供需关系：
            当供大于需时，客户通常更容易谈判价格，因为供应商争相吸引客户。
            相反，如果需求大于供应，供应商则可能更容易维持价格，因为客户可能会面临产品或服务短缺的风险。
        产品或服务的可替代性：
            如果产品或服务有可替代的选项，客户更容易谈判价格，因为他们可以选择其他供应商或产品。
            如果产品或服务是独特的或有高度专业化的，供应商可能具有更大的议价能力。
        客户的购买量：
            大型客户通常拥有更大的议价能力，因为他们的购买量对供应商来说非常重要。
            小型客户可能会受到供应商更少的关注，因此议价能力较低。
        供应商的品牌声誉：
            供应商的品牌声誉和信誉可以影响客户的议价能力。知名供应商可能更容易维持价格，因为客户信任他们的产品或服务。

        交易的重要性：
            如果交易对客户非常重要，他们可能更倾向于谈判价格和条件，以确保获得最佳交易。
            对于较小或次要的交易，客户可能会减少谈判的程度。
        客户的谈判技能：
            客户的谈判技能和策略也是议价能力的关键因素。一些客户可能更擅长谈判，而其他客户可能较为 passively 接受供应商的条件。
        市场透明度：
            市场信息的透明度可以影响客户的议价能力。如果客户了解市场价格和竞争情况，他们可能更有信心进行谈判。
